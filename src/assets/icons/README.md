# トレイアイコン仕様

このディレクトリには、システムトレイに表示されるアイコンファイルを配置します。

## 必要なアイコンファイル

### 1. tray-idle.png
- **用途**: アイドル状態（録音していない）
- **推奨サイズ**: 16x16px または 32x32px
- **形式**: PNG（透過背景推奨）
- **デザイン**: グレー系のマイクアイコンなど

### 2. tray-recording.png
- **用途**: 録音中状態
- **推奨サイズ**: 16x16px または 32x32px
- **形式**: PNG（透過背景推奨）
- **デザイン**: 赤色のマイクアイコンやREC表示など

### 3. tray-paused.png
- **用途**: 録音一時停止状態
- **推奨サイズ**: 16x16px または 32x32px
- **形式**: PNG（透過背景推奨）
- **デザイン**: 黄色のマイクアイコンや一時停止マークなど

## プラットフォーム別推奨サイズ

### Windows
- 16x16px（標準DPI）
- 32x32px（高DPI）

### macOS
- 16x16px（標準Retina）
- 32x32px（Retina）

## フォールバック動作

アイコンファイルが存在しない場合、TrayServiceは自動的にアプリケーションのデフォルトアイコンを使用します。そのため、アイコンファイルがなくてもアプリは正常に動作します。

## アイコン作成ツール

以下のツールを使用してアイコンを作成できます：

- **Figma**: https://www.figma.com/
- **Canva**: https://www.canva.com/
- **GIMP**: https://www.gimp.org/
- **Photoshop**: https://www.adobe.com/products/photoshop.html
- **Online Icon Generator**: https://favicon.io/

## アイコン配置後の確認

アイコンファイルを配置した後は、アプリを再起動してシステムトレイに正しく表示されることを確認してください。
